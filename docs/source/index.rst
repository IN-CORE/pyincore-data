pyIncore-data modules
====================

**pyIncore-data**  is a Python data manipulation and handling package for **pyIncore** module. It allows users
to acguire and handling the data IN-CORE analyses.

To **update** pyIncore-data run the following command:

.. code-block:: bash

    conda update -c in-core pyIncore-data
